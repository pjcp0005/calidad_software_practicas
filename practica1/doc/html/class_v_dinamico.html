Detailed Description

template<class T>
class VDinamico< T >

Vector dinámico genérico con redimensionamiento automático basado en potencias de 2. Gestiona memoria dinámica propia: crece al doble cuando se llena y se reduce a la mitad cuando el tamaño lógico cae a un tercio del físico.
